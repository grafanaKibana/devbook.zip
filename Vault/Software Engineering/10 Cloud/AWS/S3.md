# Intro

Amazon S3 is an object storage service.

Common concerns: bucket naming, IAM policies, encryption, lifecycle rules, versioning, and public access settings.

## Example

List buckets and list objects in a bucket:

```bash
aws s3 ls
aws s3 ls s3://my-bucket/
```
