# Intro

Graph structures model relationships between objects. The main question is not only how nodes are stored, but which relationship operations must be cheap: neighbor traversal, reachability, connectivity, shortest paths, or component merging.

`Graph` covers explicit vertices and edges. `Disjoint Set` is not a graph representation, but academically it belongs near graph structures because it answers dynamic connectivity questions over a set of vertices.
