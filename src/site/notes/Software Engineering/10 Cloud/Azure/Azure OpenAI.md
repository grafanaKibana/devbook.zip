# Intro

Azure OpenAI is the Azure-hosted way to deploy and call OpenAI models with enterprise controls (RBAC, private networking, policy/guardrails, quotas) and integration with Azure AI Foundry.

- [Work with chat completions models](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/chatgpt)
- [Prompt engineering techniques (temperature and top_p)](https://learn.microsoft.com/azure/ai-foundry/openai/concepts/prompt-engineering)
- [Customize a model with fine-tuning](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning?view=foundry-classic)
- [Azure OpenAI REST API reference (chat completions)](https://learn.microsoft.com/azure/ai-foundry/openai/reference#chat-completions)
