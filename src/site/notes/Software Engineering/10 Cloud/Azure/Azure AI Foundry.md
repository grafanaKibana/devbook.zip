# Intro

Azure AI Foundry (Microsoft Foundry) is the Azure platform surface for building, evaluating, and operating generative AI applications (projects, models/endpoints, evaluators, traces, and guardrails).

It provides:

- A place to manage deployments/endpoints and connect apps securely.
- Built-in evaluation and observability workflows.
- Guardrails (policy controls, prompt shields, groundedness filters).
