# Intro

AI is a broad field focused on building systems that perform tasks associated with human intelligence. Machine learning (ML) is a subset of AI where models learn patterns from data to make predictions or decisions.

Key distinctions you should be able to explain:

- Training vs inference
- Supervised vs unsupervised vs reinforcement learning
- Generalization vs overfitting
- Classification vs regression

## Example

Confusion matrix terms for a binary classifier:

```text
TP: predicted positive, actually positive
FP: predicted positive, actually negative
TN: predicted negative, actually negative
FN: predicted negative, actually positive

precision = TP / (TP + FP)
recall    = TP / (TP + FN)
```
